# Node-Weather

![image](https://github.com/Jcanno/node-weather/blob/master/weather.gif)   

### 下载运行

```
git clone https://github.com/Jcanno/Node-Weather.git
cd node-weather
// 安装依赖
npm install
// 运行
npm run dev
// 需手动浏览器打开index.html
```

### 通过输入地区名称来查询当地气温，支持地区模糊查询。数据来源[中国天气网](http://www.weather.com.cn/)
